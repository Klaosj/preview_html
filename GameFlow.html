📋 Detailed Flow

⛏️ Farm Area (Safe Zone)

  • 1เดินเก็บ Crystal Node รอบๆ ถ้ำ
  • 2กด E เพื่อ Mine Crystal แต่ละก้อน
  • 3Crystal สะสมใน Pending Coins
  • 4เดินไปแตะ Sell Pad เพื่อแลกเป็น Coins
  • 5ซื้อ Upgrade Pickaxe ที่ Shop
  • 6เมื่อพร้อม — เดินเข้า Entry Door

⚔️ Combat Room

  • 1Wave 1: มอนลูกน้องโผล่ (5–12 ตัว)
  • 2ใช้ Pickaxe คลิกตีมอนในระยะ
  • 3ฆ่ามอนครบ Wave 1 → Wave 2 เริ่ม
  • 4Wave 2: มอนเยอะขึ้น, แข็งขึ้น
  • 5ฆ่ามอนครบ Wave 2 → Boss โผล่
  • 6มอนตาย = Drop Coins ทันที

👑 Boss Fight

  • 1Boss มี HP สูงมาก + ดาเมจหนัก
  • 2Boss ไล่ตาม Player และโจมตีอัตโนมัติ
  • 3HP ของ Player จะค่อยๆ ลด ถ้าโดนตี
  • 4ถ้า HP = 0 → Respawn ที่ Farm Area
  • 5ฆ่า Boss สำเร็จ → Exit Door เปิด
  • 6เก็บ Coin Drop จาก Boss (มาก)

🏆 Floor Cleared!

  • 1Exit Door เปลี่ยนจากสีแดง → สีทอง
  • 2เดินเข้า Exit Door → ไป Floor ถัดไป
  • 3Crystal ดีขึ้น, มูลค่าสูงขึ้น
  • 4มอนแข็งขึ้น — ต้อง Upgrade ก่อน
  • 5Farm Limit ใหม่สำหรับ Floor นี้
  • 6Loop ซ้ำ — ขุดก่อน แล้วค่อยสู้
